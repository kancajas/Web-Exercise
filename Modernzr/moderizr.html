Gracefully degrading CSS with Modernizr

In this tutorial we're going to use Modernizr to test the browser's capability to deal with some CSS3 features that we have used on this page. Then use the classes that Modernizr adds to the body tag to apply the styles that the browser can use. This will give us complete control over how the page looks it each browser without needing to target specific browsers with nasty CSS hacks or external Style sheets.

Features we're testing for:

  • Font Face
  • Border Image
  • Box Shadow
  • CSS Columns
  • CSS Gradients
  • CSS Reflections
  • CSS Rounded Corners
  • CSS Transforms
  • Multiple Backgrounds
  • Text Shadow
